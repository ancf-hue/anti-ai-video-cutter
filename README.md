# 🎬 Anti-AI Video Cutter

**Бесплатное офлайн-приложение для обрезки видео с наложением текста**

Полностью работает в браузере без интернета. Ваши видео никуда не отправляются — вся обработка происходит локально на вашем устройстве.

## ✨ Возможности

- 🎥 **Обрезка видео** по таймлайну с точностью до 0.1 секунды
- ✏️ **Наложение текста** с настройкой цвета, размера, шрифта и обводки
- ✋ **Перетаскивание текста** мышью или пальцем в любое место кадра
- ✂️ **Обрезка кадра (crop)** — удаление логотипов, рамок, лишних частей
- 🔊 **Сохранение звука** при экспорте
- ⚡ **Изменение скорости** видео (0.5x, 1x, 1.5x, 2x)
- 💾 **Экспорт в WebM/MP4** прямо из браузера
- 📱 **Адаптивный дизайн** — работает на телефоне и компьютере
- 🔒 **100% офлайн** — ваши данные остаются у вас

## 🚀 Как использовать

1. Откройте `index.html` в браузере (Chrome, Firefox, Edge)
2. Загрузите видео (перетащите или нажмите кнопку)
3. Выберите нужный отрезок на таймлайне
4. Добавьте текст и настройте его положение
5. При необходимости обрежьте кадр
6. Нажмите «Экспортировать видео»

## 🛠 Технологии

- HTML5, CSS3, JavaScript (vanilla)
- Web Audio API (обработка звука)
- MediaRecorder API (запись видео)
- Canvas API (наложение текста и эффектов)
- Без внешних зависимостей и фреймворков

## 📋 Ограничения

- Максимальный размер файла: 500 МБ
- Максимальная длительность экспорта: 5 минут
- Поддерживаемые форматы: MP4, WebM, MOV
